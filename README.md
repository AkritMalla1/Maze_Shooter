### Maze Shooter

Maze Shooter is a game in first person camera perspective.
Motive of this game is to figure out the path through the maze to reach the Final boss by beating the flying monsters along the way.

- Created the 3D shooting game using Unity Game Engine.
- Created game assets:
  - Characters: Monsters, Final Boss
  - Weapons: Knife, Pistol, Assault Rifle, Sniper Rifle
  - Game UI (User Interface): Title Screen, Game Menu
  - HUD (Heads-Up Display): Mini-map, Health bar, Ammo Count, Remaining Lives
  - Game Layout: Level 1
